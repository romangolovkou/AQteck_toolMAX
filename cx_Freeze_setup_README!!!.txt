для збірки конфігуратору через cx_Freeze потрібно у консолі пайчарм викликати

python cx_freeze_setup.py build 

скрипт виконає попередню збірку і розмістить результат у корін проекту у папці cx_Freeze_Result

далі потрібно зібрати інсталлятор
у папці Build installer в ній папка cx_Freeze там запускаємо Build.bat і в папці Output з'явится ехе з новою версією інсталлятора
