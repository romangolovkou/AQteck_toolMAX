Скрипт сайтон парсер сетап компілює до біблиотеки під віндовс модуль за відносним шляхом "AQ_lib/AQ_Devices/SystemLibrary/AqAutoDetectionLibrary.py"
у випадку внесення змін до цього модуля його потрібно розмістити за вказаним шляхом і запустити цей скрипт з консолі в пайчарм.

Залежності: скрипт потребує встановленного модулу Cython у віртуальне оточення з якого стартує скрипт (потрібно встановити через рір з консолі з проектом у пайчарм)
а також потребує встановленного компілятора MSVC для С++ (Встановлюється через Visual Studio Installer/Visual Studio Build Tools, потрібно встановити галочку навпроти 
"Розробка класичних додатків С++, та встановити це розширення)

Запуск:
 потрібно запустити компілятор викликавши call "C:\Program Files (x86)\Microsoft Visual Studio\2022\BuildTools\VC\Auxiliary\Build\vcvars64.bat"
 а потім викликати запуск скрипту з аргументами: python cyton_parcer_setup.py build_ext --inplace
