#### Тимчасове рішення, треба доробити. ###

Спочатку треба зібрати проект запустивши pyinstaller з викликом AQteckModules.spec (скрипт конфігурації для pyinstaller) з терміналу пайчарм. У папці dist з'явиться зібраний проект.

Потім викликати Build.bat від імені адміністратора, у папці Output повинен з'явитися інсталлер.
